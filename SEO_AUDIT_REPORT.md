## 3. 개선 필요 사항

### 3.1 키워드별 전용 랜딩 페이지 ✅ 적용됨

**현재**: 홈 1페이지가 3개 키워드(출장마사지, 출장안마, 출장스웨디시)를 모두 타겟  
**적용**: 각 키워드별 독립 랜딩 페이지 추가

| 경로 | 타겟 키워드 | H1 |
|------|-------------|-----|
| `/massage` | 출장마사지 | 출장마사지 |
| `/anma` | 출장안마 | 출장안마 |
| `/swedish` | 출장스웨디시 | 출장스웨디시 |

- 각 페이지는 해당 키워드에 100% 집중
- long-form-content의 serviceMassage, serviceAnma, serviceSwedish 활용
- 사이트맵·내부 링크 추가 완료

### 3.2 내부 링크 강화 ✅ 적용됨

- 홈 서비스 섹션에서 출장마사지·출장안마·출장스웨디시 제목을 각 키워드 페이지로 링크

### 3.3 hero 이미지 alt ✅ 적용됨

- hero 이미지 alt에 "출장마사지 출장안마 출장스웨디시 - 고객 지정 장소로 방문하는 프리미엄 홈케어 마사지 서비스" 추가

### 3.4 추가 점검 사항

- **Google Search Console**: sitemap 제출, 색인 상태 확인
- **지역 페이지**: 일부 지역(강서구, 인천, 수원, 부천)은 출장마사지만 있고 출장안마·출장스웨디시 페이지 없음 → 추가 시 롱테일 확대
